TIENDA ONLINE DE ZAPATILLAS

DESCRIPCIÓN
Este proyecto consiste en una página web dedicada a la venta de zapatillas.
El sitio permite a los usuarios visualizar distintos modelos, conocer sus precios y explorar un catálogo de productos de manera sencilla y ordenada.
La página está pensada como una tienda virtual básica con un diseño moderno y fácil de usar.

OBJETIVO DEL PROYECTO

Crear una página web funcional para la venta de zapatillas.

Mostrar un catálogo de productos de forma clara.

Practicar el desarrollo web con tecnologías básicas.

INSTALACIÓN Y USO

Descargar o clonar el proyecto desde GitHub.

Abrir la carpeta del proyecto.

Si se utiliza Node.js, ejecutar el servidor con el comando:
node server.js

Abrir el navegador web.

Acceder a la dirección:
http://localhost:3000

FUNCIONALIDADES PRINCIPALES

Visualización de zapatillas disponibles.

Mostrar precios de los productos.

Diseño adaptable a dispositivos móviles y de escritorio.

Navegación sencilla e intuitiva.
